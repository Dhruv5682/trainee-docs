# ☁️ AZ-104 Azure Administrator Study Pack

> **Audience:** Cloud / DevOps Engineers, Azure trainees, and certification candidates
> **Focus:** AZ-104 exam domains, mental models, hands-on practice, and quick revision
> **Tip:** Start with the handbook, use the cheatsheet for final revision, and then work through the labs.

## Study Flow

1. Read the handbook once for the big picture.
2. Use the cheatsheet to compress the domains into memory.
3. Run the practice labs or simulate them in the portal/CLI.
4. Revisit the gotchas section (handbook §8) before the exam.
5. Scan the cram sheet right before starting the exam.

## Scope

This pack covers the five AZ-104 exam domains:

- **Identity and governance** — Entra ID, RBAC, Policy, Locks, Tags, Budgets
- **Storage** — Blob tiers, SAS, redundancy, lifecycle, private endpoints
- **Compute** — VMs, VMSS, App Service, Containers, ACR, ARM/Bicep
- **Networking** — NSGs, Peering, Load Balancer, Application Gateway, VPN, ExpressRoute, DNS, Network Watcher
- **Monitoring** — Metrics, Logs, KQL, Alerts, Azure Backup
