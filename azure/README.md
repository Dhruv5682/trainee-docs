## Study Flow

1. Read the handbook once for the big picture.
2. Use the cheatsheet to compress the domain into memory.
3. Run the practice labs or simulate them in the portal/CLI.
4. Revisit the gotchas section before the exam.

## Scope

This pack is built from the AZ-104 study topic list and organized around the five exam domains:

- Microsoft Entra ID and governance
- Storage
- Compute
- Networking
- Monitoring and maintenance
